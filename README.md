## Column Descriptions

* Column 1: Description of column
* Track URI: Link to the song
* Track Name: Name of the song
* Artist URI(s): Link to the artist(s) of the song
* Artist Name(s): Name of the artist who created the song
* Album URI: Link to the album the song is in
* Album Name: Name of the album the song is in
* Album Artist URI(s): Link to the first artist of the song (If there are more than one artist)
* Album Artist Name(s): Name of the first artist who created the song (If there are more than one artist)
* Album Release Date: The date the album was released
* Album Image URL: Link to the image of the album cover
* Disc Number:
* Track Number: Song number on the album
* Track Duration (ms): Length of the song in miliseconds
* Track Preview URL: A few second preview of the song
* Explicit: Whether the not the song is explicit
* Popularity: Rank of the song from 0 to 94
* ISRC:
* Added By: Who added it to the dataset
* Added At: Time added to the dataset
* Artist Genres: The songs artist genres
* Danceability: How danceable the song is from 0 to 0.99 (the greater the number, the more danceable)
* Energy: The energy of the song from 0 to 1
* Key: The key of the song from 0 to 11
* Loudness: The loudness of the song from -29.4 to 2.77 (the greater the number, the louder the song)
